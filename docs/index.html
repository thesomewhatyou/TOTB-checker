üöÄ Quick Start

Get the bot up and running in a few simple steps:

  1. Prerequisites: Python 3.8+, a Discord Bot Token, and a Discord Channel ID. A Role ID for pings is optional.
  2. Clone the Repository: git clone https://github.com/USERNAME/REPOSITORY_NAME # Replace! cd dandys-world-bot
  3. Install Dependencies: pip install -r requirements.txt

    (Or use `uv add discord.py aiohttp beautifulsoup4 python-dotenv trafilatura` if you prefer UV)

  4. Environment Setup: Create a .env file in the project root: 
    DISCORD_BOT_TOKEN=your_bot_token_here
DISCORD_CHANNEL_ID=your_channel_id_here
PING_ROLE_ID=your_role_id_here # Optional
PING_USER_IDS=user1_id,user2_id # Optional, comma-separated
LOG_LEVEL=INFO # Optional, defaults to INFO
  5. Run the Bot: python bot.py

For detailed Discord Application setup (creating bot, permissions, invite link), please refer to the README.md in the repository.

‚öôÔ∏è How It Works

Monitoring Schedule

  • Regular Checks: Every hour, precisely on the hour (e.g., 17:00, 18:00 UTC).
  • Midnight UTC Special: Waits an additional minute after 00:00 UTC for potential board updates before checking.
  • Smart Announcements: Only notifies if the character has genuinely changed or the board timer indicates a reset.

Data Sources

  • Primary Data: Dandy's World Daily Twisted Board Wiki
  • Images: Character images are extracted directly from the wiki pages.

Announcement Format

The bot sends rich Discord embeds that include:

  • Current twisted character name
  • Character image (if available)
  • Reason for the update (character change or timer reset)
  • A direct link to the wiki source page
  • Pings for configured roles/users
