üöÄ Get Started Quickly!

Want this bot for your own Dandy's World server? It's easier than you think!

What You'll Need:

  • A computer that can run Python (most can!). If you need Python, grab it from python.org (version 3.8 or newer).
  • A "Bot Token" from Discord (think of it as your bot's secret password).
  • The "Channel ID" from Discord for where you want announcements.

Easy Setup Steps:

  1. Download the Bot Files: Head to the bot's GitHub page. Click the green "Code" button, then "Download ZIP." Unzip it on your computer.
  2. Install a Few Tools: Open a command prompt (or Terminal on Mac/Linux) in the bot's folder. Type: pip install -r requirements.txt and press Enter.
  3. Create Your Settings File:
    • Find the file named .env.example in the bot's folder.
    • Make a copy and rename it to just .env (yep, starts with a dot!).
    • Open .env with a text editor (like Notepad).
    • Fill in your DISCORD_BOT_TOKEN and DISCORD_CHANNEL_ID. You can also add optional role/user IDs to ping.
  4. Run the Bot: In your command prompt/terminal, type: python bot.py and hit Enter!

Need help creating your bot on Discord and getting the Token/Channel ID? The README file on GitHub has a detailed step-by-step guide for this tricky part!

‚öôÔ∏è How It Works (The Magic!)

So, what's this bot doing behind the scenes?

  • Checks the Clock: Every hour, on the hour, it visits the Dandy's World Wiki's Twisted Board page.
  • Midnight Smarts: It knows the update around midnight UTC (often 8 PM EST) is important, so it waits an extra minute to make sure it gets the very latest info.
  • No Spam Policy: It only sends an alert if the Twisted character is actually new, or if the board's timer has reset.
  • Info Getter: It carefully reads the page to find the character's name, their picture, and any info about when the next change is due.
  • Pretty Posts: It then crafts a nice-looking message for your Discord channel with all these details. If you've told it to, it'll also ping any roles or specific friends you listed!
